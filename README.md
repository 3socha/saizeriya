# saizeriya

Print menu of Saizeriya at random.

## Installation

```sh
$ curl -O https://raw.githubusercontent.com/horo17/saizeriya/master/saizeriya
$ install -m 755 saizeriya /usr/local/bin/saizeriya
$ rm saizeriya
```

## Usage

```sh
$ saizeriya
半熟卵のミートソースボロニア風
```

## Uninstall

```sh
$ rm /usr/local/bin/saizeriya
```
